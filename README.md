\# 🏨 Smart Hostel Management System



A full-stack \*\*Hostel Management System\*\* designed to manage hostel operations such as student registration, room allocation, complaints, fee payments, and digital gatepass management.



The system consists of a \*\*React Frontend\*\* and a \*\*Spring Boot Backend REST API\*\*.



---



\# 🚀 Features



\### 👨‍🎓 Student Management



\* Student registration

\* Student profile management

\* Hostel admission records



\### 🏠 Room Management



\* Room allocation

\* Room availability tracking

\* Room booking system



\### 💰 Fee Management



\* Hostel fee records

\* Payment tracking

\* Fee status management



\### 📝 Complaint System



\* Students can raise complaints

\* Admin can assign and resolve complaints

\* Complaint status tracking



\### 🚪 Digital Gatepass



\* Gatepass request system

\* QR code based gatepass

\* Entry/exit verification

\# ⚙️ Installation



\## 1️⃣ Clone the Repository



```

git clone https://github.com/Nikhil-Belide/Smart-Hostel-Management-System.git

cd Smart-Hostel-Management-System

```



---



\# ▶️ Run Backend (Spring Boot)



```

cd Backend

mvn spring-boot:run

```



Backend runs on:



```

http://localhost:8080

```



---



\# ▶️ Run Frontend (React)



```

cd Frontend

npm install

npm run dev

```



Frontend runs on:



```

http://localhost:5173

```
